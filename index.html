phenopype A phenotyping pipeline for Python
phenopype is a computer vision toolbox for Python to support biologists in extracting high dimensional phenotypic data from digital images. To get started, check the Quickstart and Tutorials section in the docs, and consult the docs homepage for detailled installation instructions and API-reference. Read the paper and visit the GitHub repository (package) for more details on the scope of the package, and browse the gallery to see real world examples of projects that used phenopype to extract phenotypic data from images.

If you found a bug in the package or any of these materials, please raise an issue in the main package repository.
